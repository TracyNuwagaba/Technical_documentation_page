Introduction

Javascript® (often shortened to JS) is a lightweighted, interpreted, object-oriented language with first class functions, and is best known as the scripting language for web pages but it's also used in many non-browser environments.


Javascript runs on the client side of the web, which can be used to design or program how the web pages behave on the occurrence of an event. Javascript is an easy to learn and also powerful scripting language widely used for controlling web behavior.


Contrary to the misconception, Javascript is not interpreted as Java. The basic syntax is intentionally similar to Java to reduce the number of new concepts required to learn the language. Language constructs such as if statements, for and while loops, switch, and try and catch blocks function the same.


However, Javascript can function as both a procedural and object-oriented language. Objects are created programmatically in Javascript by attaching methods and properties to the empty objects at run time as opposed to class definitions in Java. Once an object has been constructed, it can be used as a blueprint or prototype for creating similar objects.

Get Started
To get started with Javascript, open your text editor and write your first "Hello World" Javascript code in the script tags or external file: 
document.write("Hello World");
Open the file in your broswer and you will see the output.
Comments

Comments are lines of code that Javascript will intentionally ignore. They are a great way to leave notes to yourself and to other people to figure out what the code does.


There are two ways to write comments in Javascript:


Using // will tell Javascript to ignore the remainder of the text on the current line. 

// This is an in-line comment

You can make a multi-line comment beginning with /* and ending with */ 

/* This is a multi-line comment */

It is a good practice to regularly add comments to clarify the function of parts of your code.

Variables

Javascript variables are containers for storing data values. The names of variables called identifiers may not contain spaces or start with numbers but must begin with:

  • A letter (A–Z or a–z)
  • A dollar sign ($)
  • Or an underscore (_)

Javascript variables are declared in three ways:

  1. With the var keyword: This can be used to declare both local and global variables. Global variables are declared outside the function and can be accessed from anywhere in the program where as the local variables are declared within the function and can only be accessed from inside the function where they are declared. 
    var age = 24;

    Variables declared with the var keyword can be redeclared and cannot have block scope. Variables declared with var inside {} can be accessed from outside the block. 
    {
                            var x = 10;
                            // Can be accessed and used outside the scope 
                        }

  2. With the let keyword: Variables defined with the let keyword have block scope. Variables declared with let inside {} cannot be accessed from outside the block. 
    {
                            let firstName = "John";
                            // Cannot be accessed and used outside the scope 
                        }

  3. With the const keyword: Const is used to define variables that cannot be reassigned and have to be initialised with a value. Such variables are constant(read only). 
    const PI = 3.14;
                            PI = PI + 10; // This will give an error

    Declaring a variable with const is similar to let when it comes to Block scope.
    The x declared in the block in this example is not the same as the x declared outside the block: 
    var x = 10;
                            // Here x is 10
                            {
                                const x = 5;
                                // Here x is 5
                            }
                            // Here x is 10
Data Types

In programming, data types is an important concept. Javascript variables can hold many data types forexample:

  • Primitive data types: These data types have a single simple data value with no additional properties and methods. Javascript has seven primitive data types and these include numbers, booleans, strings, undefined, null, bigint and symbol.

    • A String: It is a series of characters. Strings are written with double quotes. You can use single or double quotes. Strings are used to represent textual data. 
      var name = "John Doe";
    • Numbers: Numbers can be written with or without decimals. 
      var x1 = 34;
                                var x2 = 34.00;
    • Booleans: Booleans can only have two values, true or false. Booleans are often used in conditional testing. 
      var x = 5;
                                var y = 5;
                                var z = 6;
                                (x == y) // Returns true
                                (x == z) // Returns false
    • Undefined: The Undefined variable is one without a value. 
      var car; // value and type are undefined
      An empty value has nothing to do with undefined for example: 
      var car = ""; // value is "", the typeof is "string"
    • Null: In Javascript null is nothing. It is supposed to be something that doesn't exist.
    • BigInt: The BigInt type is a numeric primitive in Javascript that can represent integers with arbitrary precision. You can safely store and operate on large integers even beyond the safe integer limit for Number data type.
      A BigInt is created by appending n to the end of the integer or by calling the constructor. 
      const x = 2n ** 53n;
                                // 9007199254740992n
    • Symbol: A Symbol is a unique and primitive value and may be used as the key of an object property.

  • Reference data types: Javascript has 3 data types that are passed by reference: Array, Function and Object. They are all technically objects and so we refer to them collectively as objects.

    • Arrays: An array is a data type that represents a collection of elements (values or variables).Arrays can store numbers, strings, objects, booleans etc. 
      var score = ["Maths", 90];
    • Objects: They are the most commonly used data type in Javascript. They are similar to arrays, except in order to access and modify data we use properties instead of indexes. Objects store data in a structured way. Objects are written as key-value pairs separated by commas. 
      var student = {
                                name: "Tracy",
                                platform: "FreeCodeCamp",
                                course: "Javascript",
                                year: 2021
                            };
    • Functions: They are one of the fundamental building blocks in Javascript. A function in Javascript is similar to a procedure that performs a task. It should take some input and return an output.
      A function declaration consists of the function keyword followed by the name of the function, a list of parameters to the function, enclosed in parentheses and separated by commas, the Javascript statements that define the function, enclosed in curly brackets {...}. 
      function square(number) {
                                return number * number;
                            }
Conditional Statements

Conditional statements are used to make decisions in code. A conditional statement is a set of commands that executes if a specified condition is true. Javascript supports two conditional statements: if...else and switch


Use the if statement to execute a statement if a logical condition is true. Use the else clause to execute a statement if the condition is false for example: 

if (condition) {
                    statement_1;
                } else {
                    statement_2;
                }

You can also compound the statements using else if to have multiple conditions tested in sequence. 

if (condition_1) {
                    statement_1;
                } else if (condition_2) {
                    statement_2;
                } else if (condition_3) {
                    statement_3;
                } else {
                    statement_last;
                }

Switch statement selects one of many code blocks to be executed. 

switch(expression) {
                    case x:
                        // code block
                        // break;
                    case y:
                        // code block
                        // break;
                    default:
                        //code block
                    }
Javascript Operators

Javascript has the following type of operators:

  • Assignment Operator. It assigns a value to its left operand based on the value of its right operand. 
    var x = 10;

    There are also compound assignment operators for example:
    • Addition Assignment(+=)
    • Subtraction Assignment(-=)
    • Multiplication Assignment(*=)
    • Division Assignment(/=)
    • Remainder Assignment(%=)
    • Exponentiation Assignment(**=)

  • Arithmetic Operators. They are used to perform arithmetic on numbers. They take numerical values as their operands and return a single numerical value. The standard arithmetic operators are addition(+), subtraction(-), multiplication(*) and division(/). In addition to the standard arithmetic operators, Javascript provides the following arithmetic operators.
    • Remainder(%). A binary operator that returns the integer remainder of the two operands 
      12 % 5 // returns 2
    • Increment(++). A unary operator that adds one to its operand. If used as a pre-increment(++x), returns the value of its operand after adding one. If used as a post-increment (x++), returns the value of its operand before adding one.
    • Decrement(--). A unary operator that substracts one from its operand. The pre-decrement and post-decrement work the same way as pre-increment and post-increment respectively.
    • Exponentiation(**). Calculates the base to the exponent power, that is baseexponent 
      2 ** 3 // returns 8

  • Comparison Operators. A comparison operator compares its operands and returns a logical value based on whether the comparison is true for example:
    • Equal(==). Returns true if the operands are equal.
    • Not equal(!=). Returns true if the operands are not equal.
    • Strict equal(===). Returns true if the operands are equal and of the same type.
    • Strict not equal(!==). Returns true if the operands are of the same type but not equal or are of different type.
    • Greater than(>). Returns true if the left operand is greater than the right operand.
    • Greater than or equal to(>=). Returns true if the left operand is greater than or equal to the right operand.
    • Less than(<). Returns true if the left operand is less than the right operand.
    • Less than or equal to(<=). Returns true if the left operand is less than or equal to the right operand.

  • Logical operators. They are typically used with boolean values and return a boolean value. However when used with a non boolean value, they may return a non boolean value for example:
    • Logical AND(&&). Returns true if both operands are true, otherwise returns false.
    • Logical OR(||). Returns true if either operand is true. If both are false, returns false.
    • Logical NOT(!). Returns false if single operand can be converted to true, otherwise returns true.
Loops

Loops offer a quicky and easy way to perform tasks repeatedly. Loops provided in Javascript are:

  1. A for loop. It repeats until a specified condition evaluates to false. 
    2. for ([initialExpression]; [conditionExpression]; incrementExpression) {
                        statement
                    }
  2. Do while loop. It is always executed once before the condition is checked. When the condition is true, the statement executes again and when it's false, the execution stops. 
    do {
                            // code block to be executed
                        } while(condition);
  3. A while loop. Executes its statement as long as a specified condition evaluates to true. If the condition becomes false, statement within the loop stops executing. 
    4. while (condition) {
                        statement;
                    }
